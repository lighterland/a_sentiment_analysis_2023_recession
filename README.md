## a_sentiment_analysis_2023_recession

This is a sentiment analysis project that aims to analyze tweets sentiment towards the potential 2023 recession. The project utilizes the `recession_tweets_oct_dec_22.csv` dataset and `a_sentiment_analysis_2023_recession.ipynb` as the source code and documentation for the analysis. The project is done by using Jupyter Notebook with following modules:
- pandas
- matplotlib
- seaborn
- nltk
- re
- string
- stopwords from nltk.corpus
- SentimentIntensityAnalyzer from nltk.sentiment.vader
- WordCloud and STOPWORDS from wordcloud
